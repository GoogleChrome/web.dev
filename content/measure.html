How your site is measured

When you measure your site, web.dev uses the Lighthouse tool. Tests are run using Lighthouse's default settings:
  • Mobile device
  • All audits below
  • Throttled to simulated fast 3G & 4x CPU slowdown
You can measure your site using different settings, like if you want to test your desktop site or turn off audits that you don't care about. To change your report settings, run Lighthouse:
  • in Chrome DevTools
  • as a Node module, or
  • with the Chrome Extension
